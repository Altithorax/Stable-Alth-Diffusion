# 🎨 Stable Alth Diffusion

Stable Diffusion with an interface inspired by the AUTOMATIC1111 UI.

## 🚀 Features

- Image generation using positive and negative prompts
- Custom model configuration
- Easy prompt editing
- Organized structure for testing and experimentation
